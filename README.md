## Descripción (Español)

*Reproductor de Video Online* es un proyecto simple que permite reproducir videos desde enlaces en línea directamente en la interfaz de la aplicación. Está diseñado como una práctica de desarrollo web y para aprender a manejar *HTML, CSS y JavaScript* en la creación de interfaces interactivas.  

*Características principales:*
- Reproducción de videos a partir de enlaces online.
- Controles básicos: reproducir, pausar y ajustar volumen.
- Interfaz minimalista y fácil de usar.
- Compatible con los navegadores modernos.

*Tecnologías utilizadas:*
- HTML5
- CSS3
- JavaScript (vanilla)
- Git / GitHub para control de versiones

*Estado del proyecto:*
- Proyecto simple / versión de práctica
- Ideal como aprendizaje y demostración de manejo básico de video en la web
- Posibles mejoras: agregar playlists, búsqueda de videos, soporte para más formatos.

---

## Description (English)

*Online Video Player* is a simple project that allows playing videos from online links directly in the app interface. It is designed as a web development practice and to learn how to handle *HTML, CSS, and JavaScript* to create interactive interfaces.

*Main Features:*
- Play videos from online links.
- Basic controls: play, pause, volume adjustment.
- Minimalist and user-friendly interface.
- Compatible with modern browsers.

*Technologies Used:*
- HTML5
- CSS3
- JavaScript (vanilla)
- Git / GitHub for version control

*Project Status:*
- Simple project / practice version
- Perfect for learning and demonstrating basic web video handling
- Possible improvements: playlists, search, more video formats support.
