<p>Es una web de reproductor de video local y privado, con una interfaz en español que permite cargar archivos desde el dispositivo, controlar la reproducción (play, pausa, avance, volumen), y acceder a funciones como pantalla completa, mini ventana y descarga. También incluye atajos de teclado para facilitar la navegación.</p>
